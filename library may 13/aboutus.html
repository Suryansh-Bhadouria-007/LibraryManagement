About Library Search System using Semantic Web

Humans are capable of using the Web to carry out tasks such as finding the translation for "twelve months", reserving a library book, and searching for the lowest price for a DVD. However, machines cannot accomplish all of these tasks without human direction, because web pages are designed to be read by people, not machines!!

The semantic web is a vision of information that can be readily interpreted by machines, so machines can perform more of the tedious work involved in finding, combining, and acting upon information on the web.

The proposed work intends to design and implement a library search system in which ontology based semantic search is performed for the queries fired by the user.

How it works

All the data for the library is stored in the form of RDF triples. These RDF triples are represented in RDF/XML. These RDF triples are stored in a database which is queried by SPARQL. User is provided with a web interface which has RDF triple store as a backend.
